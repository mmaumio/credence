A clean, lightweight and minimal WordPress theme for versatile use across every niche you can think.

== Description ==

Credence is lightweight and highly customizable Theme. 
You can create almost any type of websites including blog, agency, portfolio, photography and WooCommerce store with the help of it's User friendly & professional design. 
Not to mention it's Gutenberg compatibility for seameless blogging. 
We made sure it's responsive, RTL & translation ready. Credence follows the best SEO practices as well.
It Works with the most popular page builders like Elementor, Visual Composer etc without any issues.

== Installation ==

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "Credence"
3. Install and Activate
